✏️ What to Edit First

These files need your content:

  1. index.html - Update church details, events, services
  2. sermons.html - Add your actual sermons
  3. css/styles.css - Optional: change colors to match your brand
  4. Add images to assets/ folder for church photos

DO NOT edit (unless you know code):

  1. js/script.js - Complex JavaScript (unless you're a developer)
  2. CSS complex selectors - Use simple color/text changes only
